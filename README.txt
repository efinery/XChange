XChange
=======
XChange is Java library providing a simple and consistent API for interacting with a
diverse set of financial security exchanges including Bitcoin.

Build instructions
------------------
XChange is built with Maven 3.0.3+ (although earlier versions may work) as a reactor project.
To build everything you need, simply enter

mvn clean package

at the command line.

Dependency management
---------------------
If you want to include XChange in your projects, you will need to reference one of the
Maven repositories. These are listed in the parent pom.xml. Typically you would reference the
